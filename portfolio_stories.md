# 포트폴리오 면접용 핵심 스토리 정리

면접 및 포트폴리오 이력서에 바로 복사-붙여넣기 하여 활용할 수 있는 핵심 성과 3가지입니다.

---

## 스토리 1: 머신러닝 모델의 Data Leakage 진단 및 신뢰성 확보

**문제 (Problem)**
- K-Means로 스태킹 레이블을 생성할 때 사용한 변수(`dz`, `dist_xy`)를 Random Forest 지도학습의 피처로 그대로 사용하여 Test Accuracy 100% 도출.
- 모델이 원자의 물리적 패턴을 일반화하지 않고 클러스터링 기준을 암기하는 전형적인 데이터 누수(Data Leakage) 상태 발생.
- 순환 논리 오류(Circular Reasoning)로 인해 실제 데이터 적용 시 모델의 신뢰성을 담보할 수 없는 상황.

**해결 (Solution)**
- 비정상적인 100% 성능을 의심하고, 데이터 누수 발생 지점을 스스로 진단하여 파이프라인 전면 재설계.
- 분류 기준이 된 파생 변수(`dz`, `dist_xy`)를 훈련 피처에서 완벽히 배제 및 통제.
- 개별 원자의 순수한 3차원 공간 좌표(`x, y, z`) 정보만을 입력값으로 사용하는 공정한 'Physics-Aware Model'을 구축.

**결과 (Result)**
- 엄격하게 통제된 피처 환경에서도 모델이 원자들의 미세한 높낮이 패턴을 순수하게 학습하여 91.94%의 공정하고 신뢰할 수 있는 일반화(Generalization) 성능 입증.
- 숫자의 함정에 빠지지 않고 데이터 사이언티스트로서 실험 설계의 타당성을 비판적으로 검증하는 역량 증명.
- 이 전체 과정을 다른 연구자도 똑같이 검증할 수 있도록 Jupyter Notebook 자동 생성 파이프라인 구현.

## 스토리 2: 비정형 시뮬레이션 데이터의 지능형 파싱 파이프라인 설계

**문제 (Problem)**
- LAMMPS 시뮬레이션의 출력 파일(`dump.minimization`)은 수만 줄의 텍스트로 구성된 비정형 데이터 형태.
- 머신러닝 파이프라인에 입력하기 위해서는 이 텍스트 아웃풋을 정형화된 표(Tabular) 형태로 변환해야 함.
- 엑셀 등 수작업이나 단순 스크립트에 의존할 경우, 데이터 구조 변경 시 병목이 발생하고 파이프라인의 재현성이 떨어짐.

**해결 (Solution)**
- 파일 내의 `ITEM: TIMESTEP`, `ITEM: ATOMS` 구조를 지능적으로 패턴 매칭하는 Python 자동화 파싱 모듈 구현.
- 동적으로 시뮬레이션의 마지막 완화 프레임(Timestep)만을 추출하고, 1만 개 이상의 개별 원자 ID, 타입, 3D 좌표를 식별.
- 추출한 Raw 데이터를 Pandas DataFrame으로 매핑하여 즉각적인 Feature Engineering이 가능한 정형 데이터로 변환.

**결과 (Result)**
- 수작업이 개입할 여지 없이, Raw 텍스트에서 ML 입력 데이터로의 변환을 단 몇 초 만에 처리하는 자동화된 데이터 파이프라인 구축.
- 향후 스태킹 각도(예: 2도, 3도) 등 변인이 달라진 새로운 시뮬레이션 데이터가 들어와도 즉각 모델에 태울 수 있는 높은 확장성 확보.

## 스토리 3: 대규모 공간 데이터 연산 최적화 (알고리즘 및 자료구조 도입)

**문제 (Problem)**
- 1.08도로 뒤틀린 초격자 물리 시스템의 무아레 패턴을 분석하기 위해서는 파편화된 상/하층 11,164개 원자들 간의 1:1 쌍(Pair) 매칭이 필수적.
- 쌍을 매칭하여 수직/수평 이격 거리를 계산할 때, 브루트포스(Brute-force) 방식의 이중 반복문을 사용하면 엄청난 연산 부하($O(N^2)$) 발생.
- 원자 개수가 늘어나는 대규모 시뮬레이션에서는 심각한 성능 병목 우려.

**해결 (Solution)**
- 수학적, 공간적 데이터 매칭의 비효율성을 해소하기 위해 SciPy의 공간 탐색 특화 자료구조인 `cKDTree (K-Dimensional Tree)` 도입.
- 하층부 5,500여 개 원자들의 2차원 공간 좌표(`x, y`)를 바탕으로 공간 트리를 빌드하고 상층부 원자 좌표를 일괄 쿼리(Query)하는 구조 설계.
- 거리 계산 알고리즘 복잡도를 O(N log N)으로 획기적으로 낮춰, 수천 단위의 원자 쌍을 최단 거리 기준으로 즉시 판별.

**결과 (Result)**
- 상-하층 원자의 수직/수평 변위를 머신러닝의 핵심 피처로 추출해내는 과정의 연산 시간을 기하급수적으로 단축 및 최적화.
- 단순한 데이터 핸들링을 넘어, 시간 복잡도를 고려한 알고리즘(Tree-based Search) 적용으로 대규모 공간 데이터 전처리 능력을 성공적으로 어필.
